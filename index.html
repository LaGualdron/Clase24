Bootstrap en el desarrollo web es altamente beneficioso debido a su capacidad para simplificar y agilizar el proceso de creación de sitios web y aplicaciones. Al ser un framework de CSS y JavaScript de código abierto, ofrece una amplia gama de componentes predefinidos que permiten a los desarrolladores construir interfaces de usuario atractivas y funcionales con menos esfuerzo y tiempo.

Bootstrap puede impactar negativamente el rendimiento de un sitio web debido a que incluye una gran cantidad de CSS y JavaScript, lo cual puede ralentizar la carga de la página al cargar recursos adicionales que podrían no ser necesarios.

Nuestra experiencia con Bootstrap fue positiva debido a que todos somos primerizos en el tema de programación y esta herramienta nos brindó un esquema o línea que nos facilitó crear una página con elementos que mejoran la experiencia del usuario.
